# Kyonggi-SmartMirror-Project
> 경기대학교 컴퓨터공학과 산학협력 프로젝트 (2026)
> 컴퓨터 비전과 AI를 활용한 실시간 면접 태도 교정 및 피드백 솔루션

#팀원
- 권휘진 (Team Leader), 양태현: 프론트엔드 UI/UX 및 하드웨어 조립
- 조준범, 구창헌, 곽태호: 백엔드 아키텍처
- 서해준, 여강록: AI 데이터 파이프라인 구축, 컴퓨터 비전(MediaPipe) 알고리즘 개발

#Tech Stack
- Language: Python 3.10+
- AI/CV: MediaPipe, OpenCV, TensorFlow
- Backend: FastAPI or Flask
- Database: MongoDB
- Hardware: Raspberry Pi / Mini PC, Half-Mirror

#Project Timeline
- 1-2주차: 환경 구축 및 하드웨어 수급
- 3-5주차: 핵심 AI 모듈 및 DB 연동
- 6-7주차: 시스템 통합 및 테스트
- 8주차: 최종 보고서 및 시연
